# 🧭 파주길잡이 (paju-giljabi)

파주시 관내 출장 동선을 자동으로 정리하고, **네이버지도·카카오맵 무료 보조 확인** 기능을 제공하는 내부 업무용 웹앱입니다.

## 실행
`index.html` 을 브라우저로 열기만 하면 됩니다. 빌드·서버·API 키가 필요 없습니다.
GitHub Pages 사용 시: `https://smile7962.github.io/paju-giljabi/`

## 주요 기능
- 출장지 입력 (출발/복귀 = 파주시청 고정, 경유 현장 **최대 15곳**)
- 출장 순서 계산(임시: 입력 순서 기준) · 도착 예정시각 · 총 소요/거리 요약
- 결과표 + 행별 **지도앱 메뉴**(네이버 열기 / 카카오 열기 / 이전→현재 구간 / 주소 복사)
- 하단 **지도앱으로 확인** 섹션: 네이버·카카오 위치 확인, 출장 순서 복사, 모바일 확인용 텍스트
- 엑셀(CSV) 저장 · 인쇄

## 무료 사용 원칙 (중요)
지도앱 연동은 **무료 공개 URL 방식만** 사용합니다. 유료 API·키·제휴·비공식 호출·웹 스크래핑을 쓰지 않습니다.
- 위치 확인: `https://map.naver.com/p/search/{주소}`, `https://map.kakao.com/?q={주소}`
- 구간 길찾기: `https://map.kakao.com/?sName={출발}&eName={도착}` (카카오 공개 URL)
- 네이버는 무료 공개 URL이 텍스트 기반 길찾기를 안정적으로 지원하지 않아, 구간 길찾기는 카카오맵을 사용합니다.
- 다중 경유지를 한 번에 여는 기능은 무료 범위에서 불안정하여 **개별/구간 단위**로 제공합니다.

지도앱 연동은 **보조 기능**이며, 파주길잡이의 출장 순서 계산을 대체하지 않습니다.
