# 🧭 파주길잡이 (paju-giljabi)

파주시 관내 출장 동선을 자동으로 정리하고, **네이버지도·카카오맵 무료 보조 확인** 기능을 제공하는 내부 업무용 웹앱입니다.

## 실행
`index.html` 을 브라우저로 열기만 하면 됩니다. 빌드·서버·API 키가 필요 없습니다.
GitHub Pages 사용 시: `https://smile7962.github.io/paju-giljabi/`

## 주요 기능
- 출장지 입력 (출발/복귀 = 파주시청 고정, 경유 현장 **최대 15곳**)
- **도로 기반 최적 경로 계산** (VWorld 좌표변환 + OSRM 도로 경로) — 실거리·실시간·최적 방문순서
  - 인증키 미설정·오류 시 자동으로 **임시 계산(입력 순서·가정값)** 으로 폴백
- 도착 예정시각 · 총 소요/거리 요약
- 결과표 각 순서 행의 **지도 버튼**: `네이버지도`(위치) / `카카오맵`(위치) / `이전→여기 길찾기`(카카오)
- 엑셀(CSV) 저장 · 인쇄

## 도로 기반 최적 경로 설정 (지오코딩)
도로 기준 최적순서·실거리·실시간 계산은 **키 없이 바로** 동작합니다(OpenStreetMap). 한국 주소 정확도를 높이려면 카카오 JS 키를 선택적으로 넣습니다.

| 단계 | 사용 서비스 | 키 | 비고 |
|---|---|---|---|
| 주소 → 좌표 (기본) | OpenStreetMap (Nominatim) | **불필요** | 브라우저 직접 호출 OK, 정확도 보통, 1초 1요청 정책 준수(자동 지연) |
| 주소 → 좌표 (정확도↑) | 카카오 지도 JS SDK | 카카오 JS키(무료) | 한국 주소 정확. 도메인 등록 필요 |
| 도로 거리·시간 + 최적순서 | OSRM `/trip` 공개 서버 | 불필요 | 자동차 도로 기준 추정 |

### 카카오 JS 키(선택) 사용 시
1. https://developers.kakao.com → 애플리케이션 추가 → **JavaScript 키** 복사
2. 앱 → **플랫폼 → Web → 사이트 도메인**에 접속 주소 등록 — 예: `https://smile7962.github.io` (로컬은 `http://localhost`)
3. 앱의 **⚙️ 도로 기반 최적 경로 설정**에 JS키 붙여넣기 (키는 **이 브라우저(localStorage)에만** 저장, 서버 전송 없음)
4. **🔌 연결 테스트**로 정상 여부 확인

> ⚠️ **VWorld 지오코더는 사용하지 않습니다.** VWorld REST API는 브라우저 직접 호출(CORS)을 허용하지 않아(도메인을 맞춰도 차단됨) 순수 정적 웹앱에서는 쓸 수 없습니다. 꼭 VWorld를 써야 한다면 별도 서버 프록시(예: Cloudflare Worker)가 필요합니다.
> 거리·시간은 자동차 도로 기준 추정치이며 실제 교통상황과 다를 수 있습니다. 공개 서버(OSRM/Nominatim)는 소량·보조 용도로만 사용하세요.

## 무료 사용 원칙 (중요)
지도앱 연동은 **무료 공개 URL 방식만** 사용합니다. 유료 API·키·제휴·비공식 호출·웹 스크래핑을 쓰지 않습니다.
- 위치 확인: `https://map.naver.com/p/search/{주소}`, `https://map.kakao.com/?q={주소}`
- 구간 길찾기: `https://map.kakao.com/?sName={출발}&eName={도착}` (카카오 공개 URL)
- 네이버는 무료 공개 URL이 텍스트 기반 길찾기를 안정적으로 지원하지 않아, 구간 길찾기는 카카오맵을 사용합니다.
- 다중 경유지를 한 번에 여는 기능은 무료 범위에서 불안정하여 **개별/구간 단위**로 제공합니다.

지도앱 연동은 **보조 기능**이며, 파주길잡이의 출장 순서 계산을 대체하지 않습니다.
